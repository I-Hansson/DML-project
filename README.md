# DML-project


This project resolves around trying to fine-tune GPT2 for generating better song lyrics for differnt genres.

In the preprocessing folder, we will do all preprocessing. Generate the data frames that is later used in the models.

In the notebook, there is the model code, and the eavluation code for the project. Also two notebooks for running inference on the two models

You will need to download the data set yourself from this kaggle page: https://www.kaggle.com/datasets/neisse/scrapped-lyrics-from-6-genres
